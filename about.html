About R3C

R3C, which stands for resistor color code calculator, was created on March 13th, 2014. It is a simple tool for those who want to calculate how many ohms of resistance a resistor provides. R3C is a 100% free tool with an open source, located here. Thanks for using, don't forget to be awesome!
